# AI agent using Akeneo PIM's MCP server

This is a PoC to test the capabilities of AI with MCP server.

## Requirements

- node and npm
- openAI API key

## Usage

Install dependencies

```bash
npm install
```

Create `.env` file and fill your openAI API key

```bash
cp .env.dist .env
```

Run the server (here in dev mode)

```bash
npm run dev
```
